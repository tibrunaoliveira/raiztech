# 🌱 IrrigaTech API

Sistema IoT para monitoramento e irrigação inteligente de plantas utilizando Java Spring Boot e MySQL.

## 🚀 Tecnologias
- Java
- Spring Boot
- MySQL
- JPA/Hibernate

## 📦 Funcionalidades
- Cadastro de usuários
- Cadastro de plantas
- Leituras de sensores
- Controle de irrigação

## 🔗 Endpoints

GET /usuarios
POST /usuarios

GET /plantas
POST /plantas

GET /leituras
POST /leituras

GET /irrigacoes
POST /irrigacoes
